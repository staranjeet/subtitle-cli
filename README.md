Subtitli
=====

This python package finds and downloads the subtitle for a movie or a videos.
Subtitli stands for subtitle-cli

Install
=======

* Using `pip`
```
$ pip install subtitli
```

* From source

```
$ git clone https://github.com/staranjeet/subtitli
$ cd subtitli
$ python setup.py install
```

Usage
=====

```
$ subtitli <movie>

```
